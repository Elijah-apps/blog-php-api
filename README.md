# Blog API with Leaf PHP

This is a simple Blog API built with Leaf PHP. The API allows you to manage blog posts and users.

## Features

- Post Management (CRUD)
- User Registration and Login

## Prerequisites

- PHP 7.4+
- Composer
- MySQL
